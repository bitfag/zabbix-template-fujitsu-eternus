# zabbix-template-fujitsu-eternus

This is a zabbix template to monitor SAN Fujitsu Eternus.

## Tested with

* Eternus DX100

## Items

* Overall system status
* Autodiscovered metrics for each volume:
  * IOPS
  * Throughput
  * Response time
  * Cache hit rate

## Triggers

* Overall system status

## Graphs

* Each performance metric has it's own nice graph
